# Spacecraft Sensor Simulator

A Node.js module for simulating spacecraft sensor readings and mineral discoveries. Perfect for space exploration 
applications, educational projects, or any simulation requiring realistic sensor data.

## Features

- Thermal sensor simulation (-100°C to 150°C)
- Radiation level readings (0-1000 mSv)
- Proximity detection (0-1000 meters)
- Random mineral discovery simulation
- Configurable mineral findings
